# TripNest

TripNest is a full-stack web application that allows users to explore property listings, create their own listings, and share reviews. The application includes secure user authentication, authorization, and full CRUD functionality for listings and reviews.

---

## Features

- User Signup, Login, and Logout
- Create, Edit, and Delete Listings
- View all listings and individual listing details
- Add and Delete Reviews
- Secure Authentication using Passport.js
- Authorization (Only owners can modify their listings)
- Flash messages for user feedback
- Session management using MongoDB
- RESTful routing

---
